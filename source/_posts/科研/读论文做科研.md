# 如何读论文-李沐

- 论文结构
  - 1. titile
    2. abstract
    3. intro
    4. method
    5. experiment
    6. conclusion
- 花三遍读一篇论文
  - 1. 读标题
    2. 读摘要
    3. 读结论（通常和摘要是一样的，摘要中可能会提出一些问题，结论用实际的结论和数字证明）
    4. 看一下实验部分的图和表
  - 读整个文章，从标题到最后，主要看图和表
  - 精读

# 如何写论文-李沐

- 用有新意的方法有效的解决一个研究问题
- 摘要
  - 我要解决什么样的研究问题
  - 我的方法的新意在哪里
  - 最后我的结果怎么样
- 新意度 X 有效性 X 问题大小 = 价值
